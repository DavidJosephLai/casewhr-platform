# 🏦 永豐銀行寰宇金融 API - 快速申請指南

## 📋 申請時需要詢問的具體 API 名稱

與永豐銀行聯繫時，請明確說明需要以下 API 服務：

### ✅ 核心 API（必須）

1. **企業批次代付 API**（或稱「批次轉帳 API」）
   - 英文名稱：Batch Payment API / Batch Transfer API
   - 用途：自動撥款給接案者
   - 需求：支援台幣（TWD）跨行/同行轉帳

2. **帳戶餘額查詢 API**
   - 英文名稱：Account Balance Inquiry API
   - 用途：即時查詢平台帳戶餘額

3. **交易狀態查詢 API**
   - 英文名稱：Transaction Status Query API
   - 用途：查詢轉帳進度和結果

### ⭐ 額外建議申請（可選）

4. **帳戶驗證 API**（強烈建議）
   - 英文名稱：Account Verification API
   - 用途：驗證收款帳號正確性
   - 好處：減少轉帳失敗率

5. **交易通知 Webhook**（強烈建議）
   - 英文名稱：Transaction Notification Webhook
   - 用途：轉帳完成時自動通知平台
   - 好處：即時更新提現狀態

6. **交易明細查詢 API**
   - 英文名稱：Transaction History API
   - 用途：定期對帳和財務報表

## 💰 預估費用（供參考）

### 開戶費用
- 企業帳戶開戶：通常免費
- 網路銀行企業版：可能需要年費（約 NT$3,000-10,000）

### API 服務費
- 可能的收費模式：
  - **月費制**：每月固定費用（例如：NT$5,000-20,000）
  - **按筆計費**：每筆轉帳收取費用（例如：NT$5-15/筆）
  - **混合制**：月費 + 超量按筆計費

### 轉帳手續費
- 同行轉帳（永豐→永豐）：NT$0-10/筆
- 跨行轉帳（永豐→其他銀行）：NT$15-30/筆

**實際費用以永豐銀行報價為準**

## ⚠️ 常見問題預防

### Q1: 永豐銀行會不會說「沒有 API 服務」？

**預防措施：**
- 先說明是「企業客戶」
- 強調需要「批次代付」或「自動化轉帳」
- 如果客服不了解，要求轉接「企業金融部門」或「數位金融部門」

### Q2: 如果永豐銀行沒有公開的 API 怎麼辦？

**替代方案：**
1. 詢問是否有「檔案傳輸」方式（例如：上傳 CSV 批次轉帳）
2. 或使用永豐銀行的合作夥伴（金融科技公司）的 API
3. 考慮其他銀行：
   - 玉山銀行（您之前考慮的）
   - 國泰世華銀行
   - 中國信託銀行

### Q3: 審核不通過怎麼辦？

**可能原因：**
- 公司成立時間太短
- 預估交易量不足
- 風險控管不夠完善

**解決方式：**
- 提供更詳細的營運計畫
- 說明平台的 KYC 和防詐騙機制
- 先從小額度開始（例如：每日限額 NT$50,000）
