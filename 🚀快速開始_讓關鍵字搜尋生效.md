## ✅ **3 分鐘快速設置**

### **第 1 步：在 OpenAI Platform 獲取 API Key** ⏱️ 1 分鐘

**在您現在打開的網頁上：**

1. **點擊左側選單的「API keys」** 
   ```
   左側邊欄 → API keys（鑰匙圖標）
   ```

2. **點擊「+ Create new secret key」按鈕**
   - 輸入名稱：`CaseWHR-Production`
   - 點擊 **Create secret key**

3. **⚠️ 立即複製 API Key！**
   - API Key 格式：`sk-proj-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
   - **只會顯示一次**，立即複製！

---

### **第 2 步：配置到 Supabase** ⏱️ 1 分鐘

1. **打開新分頁，前往：**
   ```
   https://app.supabase.com/project/YOUR_PROJECT_ID/settings/functions
   ```

2. **添加環境變數：**
   - 點擊 **"Add new secret"**
   - Name: `OPENAI_API_KEY`
   - Value: 貼上剛才複製的 API Key
   - 點擊 **Save**
