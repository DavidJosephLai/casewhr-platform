# ⚡ 快速診斷 - wismachion.com 空白

---

## 🐛 **問題：wismachion.com 顯示空白**

---

## 🔍 **3 步快速診斷：**

### **步驟 1：檢查 DNS**

**在線查詢：**
```
訪問：https://dns.google/

搜索：wismachion.com

結果：
✅ 如果看到 IP：76.76.21.21 → DNS 已配置
❌ 如果沒有記錄 → DNS 未配置（需要設置）
⏳ 如果看到其他 IP → DNS 配置錯誤（需要修改）
```

---

### **步驟 2：檢查 Vercel**

```
訪問：https://vercel.com/dashboard

專案 → Settings → Domains

檢查：
✅ 列表中有 wismachion.com → 已添加
❌ 列表中沒有 → 需要添加

如果已添加，狀態是什麼？
✅ Valid（綠色勾勾）→ 配置正確
⏳ Pending → 等待 DNS 生效
❌ Invalid → DNS 配置有問題
```

---

### **步驟 3：檢查瀏覽器**

```
訪問：wismachion.com

F12 → Network 標籤

查看：
✅ 狀態碼 301 → Redirect 正常
✅ 狀態碼 200 → 頁面加載正常
❌ 狀態碼 404 → 域名未添加到 Vercel
❌ 狀態碼 502/503 → 服務器錯誤
❌ 狀態碼 ERR_NAME_NOT_RESOLVED → DNS 未配置
```

---

## 🎯 **根據診斷結果：**

### **結果 A：DNS 未配置**

**症狀：**
```
✅ dns.google 查不到記錄
✅ 瀏覽器顯示空白或 "找不到網站"
✅ Network 顯示 ERR_NAME_NOT_RESOLVED
```

**解決：**
```
1. 登入域名注冊商（GoDaddy/Namecheap/等）

2. DNS 管理

3. 添加 A 記錄：
   類型: A
   名稱: @
   值: 76.76.21.21
   
4. 保存

5. 等待 5-10 分鐘

6. 重新測試
```

---

### **結果 B：域名未添加到 Vercel**

**症狀：**
```
✅ dns.google 查到 IP：76.76.21.21
✅ 但瀏覽器顯示 404 或 Vercel 錯誤頁面
✅ Vercel Domains 列表中沒有 wismachion.com
```

**解決：**
```
1. Vercel Dashboard → Settings → Domains

2. 輸入：wismachion.com

3. 點擊 "Add"

4. 等待幾秒鐘

5. 重新測試
```

---

### **結果 C：DNS 還在傳播**

**症狀：**
```
✅ dns.google 有時查到，有時查不到
✅ 不同設備顯示不同結果
✅ Vercel 域名狀態：Pending
```

**解決：**
```
1. 等待 DNS 傳播（5 分鐘 - 2 小時）

2. 檢查傳播狀態：
   https://www.whatsmydns.net/
   輸入：wismachion.com

3. 清除瀏覽器緩存：
   Ctrl+Shift+Delete

4. 無痕模式測試：
   Ctrl+Shift+N

5. 重新測試
```

---

### **結果 D：Redirect 配置問題**

**症狀：**
```
✅ DNS 已生效
✅ 域名已添加到 Vercel
✅ 但訪問後沒有跳轉到 casewhr.com
✅ 或顯示 Case Where 首頁而不是 Wismachion
```

**解決：**
```
1. 檢查 vercel.json 是否已部署：
   Vercel → Deployments → 最新部署
   → Source → 查看 vercel.json

2. 檢查最新代碼是否已推送：
   GitHub → 倉庫 → 查看最新 commit
   
3. 如果沒有推送：
   git add .
   git commit -m "Update vercel.json redirect"
   git push origin main

4. 等待 Vercel 部署（2 分鐘）

5. 重新測試
```
